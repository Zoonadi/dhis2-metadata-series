# DHIS2 Metadata Series
This repository contains the CSV files for the metadata formats in DHIS2. The folders contain the following:

## 1. [Stop Creating Option Sets One by One](https://youtube.com/shorts/tvihb2XBrmA?si=YQrNOVkTSB7tZz2N)
- Access the [option-set](https://github.com/Zoonadi/dhis2-metadata-series/tree/main/1.%20option-set) folder in the repo to see a blank template of the Option Set CSV format and an example template of languages.

- Find the link to the Option Set format [here](https://docs.dhis2.org/en/develop/using-the-api/dhis-core-version-241/metadata.html?h=csv+metadata+import+2.41#category-option-group) or use the image below to see the structure.
